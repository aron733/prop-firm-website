# ProTradeX - Proprietary Trading Firm Website

Un site web professionnel et moderne pour une entreprise de trading propriétaire (prop firm).

## 🎯 Caractéristiques

- **Design Moderne & Responsive** : Interface épurée et adaptée à tous les appareils
- **Navigation Fluide** : Défilement lisse entre les sections
- **Sections Principales** :
  - 🏠 Accueil avec hero section impactante
  - 📊 Services (capital de trading, formation, sécurité, profit sharing)
  - 🎓 Programme détaillé en 4 étapes
  - 👥 Présentation de l'équipe
  - 📈 Statistiques clés
  - 📧 Formulaire de contact

## 📁 Structure du Projet

```
prop-firm-website/
├── index.html      # Page principale (HTML)
├── styles.css      # Feuille de styles (CSS)
├── script.js       # Fichier JavaScript (interactions)
└── README.md       # Ce fichier
```

## 🎨 Design et Couleurs

- **Couleur Primaire** : Bleu (#0066ff)
- **Couleur Secondaire** : Bleu Foncé (#001a4d)
- **Couleur d'Accent** : Orange (#ff6600)
- **Gradient** : De bleu foncé à bleu clair

## 🚀 Fonctionnalités JavaScript

- Scroll smooth vers les sections
- Animation des éléments au défilement
- Formulaire de contact interactif
- Animation des compteurs de statistiques
- Navigation sticky avec effet au scroll

## 📱 Responsive Design

Le site s'adapte parfaitement à :
- 📱 Mobiles (320px+)
- 📱 Tablettes (768px+)
- 💻 Ordinateurs (1200px+)

## 🔧 Comment utiliser

1. **Télécharger les fichiers** ou cloner le repository
2. **Ouvrir `index.html`** dans votre navigateur
3. **Personnaliser** le contenu avec vos infos :
   - Nom de l'entreprise
   - Services
   - Équipe
   - Informations de contact

## 📝 Personnalisation

### Modifier l'en-tête
```html
<div class="nav-brand">ProTradeX</div>  <!-- Votre nom -->
```

### Ajouter/Modifier les services
```html
<div class="service-card">
    <div class="icon">📊</div>
    <h3>Votre Service</h3>
    <p>Description du service</p>
</div>
```

### Ajouter des membres d'équipe
```html
<div class="team-member">
    <div class="member-avatar">AB</div>
    <h3>Votre Nom</h3>
    <p>Votre Titre</p>
    <small>Votre Bio</small>
</div>
```

## 💻 Technologies Utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Styles modernes (Flexbox, Grid, Gradients)
- **JavaScript Vanilla** : Interactions sans dépendances

## 📞 Support

Pour améliorer ce site, vous pouvez :
- Modifier les couleurs dans les variables CSS `:root`
- Ajouter de nouvelles sections
- Intégrer une API de formulaire (Formspree, EmailJS, etc.)
